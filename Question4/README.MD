# Logic Test
Solve anagram problem, and group by related anagram word.

## Input
```
['kita', 'atik', 'tika', 'aku', 'kia', 'makan', 'kua']
```

## Expected Output
```
[
  ["kita", "atik", "tika"],
  ["aku", "kua"],
  ["makan"],
  ["kia"]
]
```

## How to run?
Please execute script by:  
`node answer.js`
